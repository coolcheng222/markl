# Spring Security

## 一. 介绍

Spring Security基于Spring,提供了一套Web应用安全性完整解决方案

__认证__: 通俗认为用户能否登陆

__授权__: 判断用户有无权限

## 二. 整合springboot-helloworld

### 1. 

场景启动器

```java
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>
    
```

导入之后就直接生效了
